Описание приложения «Movie Library» (Личная кинотека)
Movie Library — это десктопное GUI‑приложение для удобного хранения и систематизации информации о фильмах. Программа позволяет вести личную коллекцию кинокартин, быстро находить нужные фильмы с помощью фильтров и сохранять данные в надёжном формате JSON.

Основные функции
Добавление фильмов: ввод информации о фильме через удобную форму с полями:

название;

жанр;

год выпуска;

рейтинг (по 10‑балльной шкале).

Отображение коллекции: просмотр всех добавленных фильмов в виде интерактивной таблицы.

Фильтрация данных: быстрый поиск по:

жанру (выпадающий список с доступными вариантами);

году выпуска (точный поиск по указанному году).

Сохранение данных: автоматическое сохранение всей коллекции в JSON‑файл при закрытии приложения.

Загрузка данных: восстановление коллекции из JSON‑файла при запуске программы.

Валидация ввода: проверка корректности данных:

год выпуска должен быть числом в разумном диапазоне (например, 1800–2030);

рейтинг должен быть числом от 0 до 10.

Интуитивный интерфейс: простой и понятный графический интерфейс на базе tkinter, доступный даже неопытным пользователям.

Технические характеристики
Язык программирования: Python 3.x.

Графический интерфейс: tkinter (стандартная библиотека Python).

Формат хранения данных: JSON (структура файла — массив объектов с полями title, genre, year, rating).

Система контроля версий: Git (для отслеживания изменений в коде и совместной разработки).

Управление зависимостями: не требуется (используется только стандартная библиотека Python).

Требования к системе
Установленный Python версии 3.6 или выше.

Операционная система: Windows, macOS или Linux.

Примеры использования
Добавление нового фильма:

Заполните поля формы: «Название» → «Интерстеллар», «Жанр» → «Фантастика», «Год выпуска» → 2014, «Рейтинг» → 8.6.

Нажмите кнопку «Добавить фильм».

Фильм появится в таблице ниже.

Фильтрация по жанру:

В выпадающем списке «Фильтр по жанру» выберите «Фантастика».

Нажмите «Применить фильтры».

В таблице отобразятся только фильмы жанра «Фантастика».

Фильтрация по году:

В поле «Фильтр по году» введите 2014.

Нажмите «Применить фильтры».

В таблице отобразятся все фильмы, выпущенные в 2014 году.

Сохранение и загрузка:

Закройте приложение — данные автоматически сохранятся в файл movies.json.

При следующем запуске коллекция загрузится из этого файла.

Преимущества
Автономность: не требует подключения к интернету.

Простота: минимум зависимостей, запускается «из коробки» после установки Python.

Надёжность: данные хранятся локально в человекочитаемом JSON‑формате.

Гибкость: код легко модифицировать и расширять (добавлять новые поля, фильтры и т. д.).

Контроль версий: использование Git позволяет отслеживать изменения в коде и безопасно вносить правки.
